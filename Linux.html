¿Qué es?

Linux es una familia de sistemas operativos open source basados en el kernel de linux , un kernel que fue lanzado el 17 de septiembre de 1991 por linus torvalds.

¿ por qué es importante aprender linux ?

Quizá en estos momentos como lector te estés preguntando ¿por qué aprender Linux?, si existen otros sistemas operativos “menos complejos” listos para que el usuario los utilice como Windows o macOS, pero nos encontramos en una época en la cual ya no podemos encerrarnos con estos sistemas ya que tienen problemas de seguridad con malwares, problemas de inestabilidad y un consumo excesivo de recursos sin contar los altos costosde licencia por utilizar estos sistemas operativos sin en cambio con alguna distribución de Linux todo esto cambia pues el tan solo hecho de la eficiencia que ofrece y de la gran comunidad de ingenieros y programadores que año con año aportan al desarrollo de las distintas distribuciones de Linux hacen que Linux sea algo mas que un simple sistema operativo si no algo único para cada persona.

¿Donde se ocupa linux?

Esto lo podemos contestar con otra pregunta ¿ Donde no se ocupa linux?, pues se encuentra presente en la actualidad practicamente en todo:

  • Celulares (Android)
  • Carros
  • Refrigeradores
  • Televisiones
  • y en sistemas de gran tamaño como lo son las supercomputadoras esto se debe practicamente a la gran eficiencia que tiene el sistema linux en cualquier hardware y el poder adaptativo que tiene este.

Comparación de linux con Windows

A continuación haremos una breve comparación de un sistema windows actual en una maquina con recursos medios y un distribución linux (opensuse) que se ejecutara en un ordenador de aproximadamente 15 años.

Recursos utilizados con opensuse

como podemos darnos cuenta, en linux a pesar de solo tener 3 gb de ram ddr2, al inicio del sistema y durante el uso del navegador abriendo unas cuantas pestañas, simplemente gasta entre 500mb a 1gb de memoria ram

Recursos utilizados con windows

Por otro lado en windows utilizando el navegador para abrir unas cuantas pestañas esta consumiendo 6gb de ram, esto muestra una mala gestión de recursos, pues la diferencia en ambos sistemas es muy grande.
